apply_properties_from_dict(abridged_data, constructions)[source]¶

Apply properties from a FaceEnergyPropertiesAbridged dictionary.

Parameters:
  • abridged_data – A FaceEnergyPropertiesAbridged dictionary (typically coming from a Model).
  • constructions – A dictionary of constructions with constructions names as keys, which will be used to re-assign constructions.
construction¶

Get or set Face Construction.

If the Construction is not set on the face-level, then it will be assigned based on the ConstructionSet assigned to the parent Room. If there is no parent Room or the the parent Room’s ConstructionSet has no construction for the Face type and boundary_condition, it will be assigned using the honeybee default generic construction set.

new_host: A new Face object that hosts these properties.
